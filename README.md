# Onboarding Checklist Builder

[![Sponsor](https://img.shields.io/badge/Sponsor-GitHub%20Sponsors-ea4aaa?logo=githubsponsors)](https://github.com/sponsors/sdotwinter)

A Python CLI to create and track onboarding checklists for new hires.

## Usage
```bash
python3 main.py init . --template backend
python3 main.py add-item "Set up 1:1 with manager" --owner sean .
python3 main.py report .
python3 main.py export . --format md --out checklist.md
```
